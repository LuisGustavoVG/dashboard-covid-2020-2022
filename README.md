# dashboard-covid-2020-2022
painel interativo construído no Power BI com dados da covid-19 do Ministério da Saúde durante o mandato do ex-presidente Jair Bolsonaro

<img width="744" height="483" alt="2026-04-03 (57)" src="https://github.com/user-attachments/assets/6bd10286-bb34-411e-9b6e-947c7e615154" />

O painel apresenta uma visão geral dos estados do Brasil mostrando os óbitos acumulados, a população total, casos acumulados e a incidência de covid-19(por 100 mil habitantes)


<img width="745" height="485" alt="2026-04-03 (19)" src="https://github.com/user-attachments/assets/437c337e-e7f7-4064-b512-ad1bf637d8c2" />

O painel possibilita que o leitor escolha a sigla do estado e ainda possa comparar com outros estados ou selecionar tudo. O leitor assim poderá executar várias comparações de dados, como podemos ver na imagem acima onde comparamos os dados dos estados do Amazonas, Ceará e Mato Grosso do Sul.
